# Admin

## O que o admin pode fazer hoje

- Listar jornadas
- Criar jornada
- Editar jornada
- Publicar ou despublicar jornada
- Listar semanas
- Criar semana
- Editar semana
- Definir semana atual
- Definir `release_at`
- Salvar `pdf_url`
- Salvar `video_url`
- Publicar ou despublicar semana
- Ver aplicações de colaboradores
- Aprovar ou rejeitar aplicações

## Áreas

- `/admin`
- `/admin/jornadas`
- `/admin/semanas`
- `/admin/colaboradores`

## Limitações atuais

- Upload real via Supabase Storage ainda não foi implementado.
- Admin ainda opera com formulários simples.
- Não há editor rico de conteúdo.

## Próximos passos sugeridos

- Upload real de PDFs e mídias.
- Melhor gestão de release por jornada.
- Melhor auditoria de publicações.
