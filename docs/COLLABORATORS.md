# Collaborators

## Propósito

A página `/colaborar` existe para discernir pessoas que desejam servir no projeto com maturidade, constância e alinhamento espiritual.

## Critérios

1. Ser cristão
2. Estar disponível para duas reuniões mensais fora do horário comercial
3. Ter clareza sobre o propósito do Reino
4. Servir com maturidade, excelência e constância

## Campos do formulário

- nome
- email
- WhatsApp
- igreja/comunidade
- cidade
- confirmação de fé cristã
- confirmação de disponibilidade
- propósito do Reino
- dons/habilidades
- motivo para participar

## Status

- `pending`
- `approved`
- `rejected`

## Fluxo

- Público envia aplicação
- Registro vai para `collaborator_applications`
- Admin lê e atualiza o status
